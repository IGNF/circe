Detailed Description

pt4d accomodates data used in single point operations: id, all type of coordinates (considering that some operations require more than one coordinate type together) and collateral projection output. The structure refers to points that have 2d or 3d geodetic coordinates and a vertical coordinate. In this version, it doesn't include velocities.
