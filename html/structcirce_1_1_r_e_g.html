Detailed Description

Structure designed to store Geodetic Frame data from Circe Legacy Metadata. Each REG instance is used to define at least 4 objects in the Circe XML metadata file: 4 related to Greenwich and other related to the native prime meridian if different from Geenwich (Greenwich-related frames are mandatory in geodetic transformations).

  • 1 or 2 geodeticReferenceSystem (id from num for internal management)
  • 1 or 2 geodeticReferenceFrame (id from num for internal management)
  • 1 GeocentricCRS for 3-D or 4-D systems (id = short_name, for use by user)
  • 1 or 2 GeographicCRS (id = short_name+'G' and id = short_name+'I'+'G' where 'I' stands for the LGO (prime meridian) name initial, for use by user)

See the resulting Circe XML metadata file for a better comprehension.
