Detailed Description

Implementation of Circe geodetic metadata writing and loading. Three metadata file formats are considered.

  • The Circe Legacy Metadata format (files usually named DataXXX.txt where XXX refers to a geographic area). It is intended to be easilly written "by hand". It can be loaded by Circe, not written.
  • The Circe v5 native XML format. It can be written from a legacy file, possibly edited "by hand", and loaded by Circe.
  • IGN.xml.
Remarks
IGNF and EPSG codes to be checked (not fully automated).
